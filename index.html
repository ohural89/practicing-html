Модуль 5. 1. При значенні css.position: relative; конкретного елементу, для інших елементів він залишаться стабільним для потоку всіх елементів.Проте, з допомогою функцій (top, right, bottom, left) його місце позиціювання можна змінити у будь яке інше місце visual port. 2. {css.position: absolute}: елемент "витягається"" з потоку всіх елементів, і допоки йому не задаються (top, right, bottom, left), він стоїть на тому ж самому місці де і стояв до того в потоці всіх елементів. Як тільки йому задаються будь які параметри з (top, right, bottom, left), абсолютий елемент починає шукати предків з відмінною позицією від static. Relative - застосувється для "виловлювання" абсолютних елементів. 3. Центрування елементів: процентами (%), при цьому елемент центрується не своїм власним центром, а своїм верхнім лівим кутом. 4. щоб центрувати елемент по його власному центру, перший спосіб: задати відємні margin, знаючи його висоту та ширину, котра найчастіше розробнику не відома, оскільки визначається самим вмістом елементу 5. Другий спосіб: через css.transform: translate (-...px / %, -...px / %) - зміщенян по х та y. У відсотках задавати краще. 6. Fixed - елемент теж "витягнений з потоку", такий елемент не шукає "предків" - він завжди позиціонується згідно viewport! 7. {css.z-index: 0 / 1 / 2 ... ;} - дає можливість змінити пріоритетнітсь відображення абсолютних елементів. 8. css.transform: translate(...px / %, ...px / %) - залежно від висоти та ширини елементу по осях x та y. При цьому, елемент залишається у потоці всіх елементів. 9. css.transform: rotate(...deg) - елемент крутиться навколо осі z в незмінній площині. 10. css.transform: scale() по осі x y. Це - масштабування, збільшення елементу, при цьому збільшується не лише сам елемент, а й і його вміст. це лише візуальна трансформація, без зміни потоку. 11. css.transform: skew(x, y) нахилення елементу (викривлення) на віссю x y 12. transform origin - для зміни осі повороту елементу

тема 10. Події. Анімації. transittion-property - яку властивість ми хочемо анімувати (фоновий колір) transittion-duration - тривалість (час) анімації/переходу (s, ms). краще у мілісекундах. Без цієї функції перехід не відбудеться також інші функції не працюватимуть. transittion-timing-function - функція розприділення часушвидкість, з якою відбувається анімація transittion-delay - затримка анімації/переходу (Див. cubic-bezier) css.transittion: property duration timing-function delay; - зазвичай так задається анімація на елемент. дана функція має зв'язок з заданими функціями hover. Ставиться на сам елемент, проте коли ставиться на ховер/фокус, не працюватиме анімація назад. значення all - спрацює для всіх функцій, через кому в тому ж самому transition для додаткових функцій.
